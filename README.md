# Tech Talks Today Website

This is a single-page website for a one-day tech talk event. It displays the schedule for the entire day, and allows users to search for talks by category.

## Technologies Used

*   **Frontend:** HTML, CSS, JavaScript
*   **Backend:** Node.js, Express

## How to Run

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sprindloo/sprindloo-event-talks-app.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd sprindloo-event-talks-app
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Start the server:**
    ```bash
    npm start
    ```

5.  **View the website:**
    Open your web browser and navigate to `http://localhost:3000`.
